# Bustin
Bustin’ is a fast-paced co-op ghost hunt game where players face mischievous spirits, manage sanity, and survive haunted manors armed with only their wits and basic gear.

# Thin Slice
Bustin’s [thin slice](https://docs.google.com/document/d/1Bdyvs7advWwhguuozBMH2WXgZ_1gd64KD4dfN07cpnU/edit?tab=t.0) covers the ghost-hunting loop with a light gear system. Players start with auto-filled basic gear (no upgrades). From their house, they choose random jobs, each leading to a random haunted manor. After success or failure, they see a rewards screen before returning home, where the only action is picking the next hunt.
